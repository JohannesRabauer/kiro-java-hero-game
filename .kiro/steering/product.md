# Product: Java Hero Cards

A gamified web application that teaches Java and Spring concepts through collectible hero cards.

## Core Concept

Players progress through sequential learning paths (BRONZE → SILVER → GOLD → SPRING_MASTER), unlocking hero cards by completing quizzes about Java/Spring topics. Each card represents a programming concept (e.g., variables, classes, streams, dependency injection).

## Key Mechanics

- **Learning Paths**: Tiered progression system with increasing difficulty and XP rewards
- **Hero Cards**: Collectible cards tied to programming concepts, unlocked sequentially within each path
- **Quizzes**: Multiple-choice questions that must all be answered correctly to unlock a card
- **XP System**: Experience points awarded per card unlock, varying by path tier (10/20/30/50)
- **Onboarding**: New player flow before entering the main learning experience

## Users

Single-player experience with a default player profile. No authentication system — designed as a learning tool rather than a competitive game.
