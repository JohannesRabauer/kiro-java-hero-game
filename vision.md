# Java Hero Cards

## Goal

Java Hero Cards teaches Java and Spring through collectible cards.

Players unlock heroes, answer questions, and complete learning paths. Each hero represents one programming concept.

Learning should feel like a game, not a course.

---

## Player Journey

### 1. Welcome

Three short animations introduce the game.

**Collect Heroes**
Learn Java by collecting cards.

**Gain Skills**
Each hero teaches one concept.

**Complete Your Deck**
Unlock every hero and master Java.

---

### 2. Roadmap

The player sees a clear learning path.

Bronze → Silver → Gold → Spring Master

Completed heroes are visible. Future heroes stay locked.

---

### 3. Learn a Hero

The next hero card appears.

Example:

**Variable Hero**
Stores information for later use.

The player sees:

* A short animation
* A simple explanation
* A Java example

Example:

```java
String name = "Alex";
```

---

### 4. Answer Questions

The player answers a few quick questions.

Example:

What does a variable do?

* Store information
* Repeat code
* Draw graphics

---

### 5. Unlock the Card

After passing the quiz, the card unlocks.

The card flips, shines, and moves into the collection.

The player earns experience points.

---

### 6. Continue

The next hero becomes available.

Each new hero builds on previous knowledge.

---

## Learning Paths

### Bronze

* Variable
* String
* Number
* Boolean
* If
* Loop
* Method

### Silver

* Class
* Object
* Constructor
* Collection
* Exception

### Gold

* Interface
* Generic
* Stream
* Lambda
* Optional
* Record

### Spring Master

* Controller
* Service
* Repository
* Bean
* Dependency Injection

Completing a path unlocks the next one.

---

## Collection

Players can open their collection at any time.

They can:

* View unlocked cards
* Flip cards
* Zoom cards
* Filter by category
* Show off completed sets

The collection should feel like a premium trading card album.

---

## Success

Players learn one concept at a time.

Every card teaches:

* What it is
* Why it matters
* How to use it

By completing the collection, players learn the foundations of Java and Spring.
